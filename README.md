# java-bank-simulator
Menu-driven Java console banking application demonstrating core programming fundamentals.

## Overview
This project is a Java console-based banking simulator designed to practice and demonstrate core programming fundamentals. The application allows a user to access a simulated bank account using a PIN and perform common banking operations through a menu-driven interface.

## Features
- PIN-based user authentication
- Menu-driven interface
- Check account balance
- Deposit funds
- Withdraw funds
- Console-based interaction using Java Scanner

## Technologies Used
- Java

## What I Learned
- Designing menu-driven programs using conditional logic
- Handling user input and validating numeric data
- Tracking and updating program state
- Debugging and improving code readability
